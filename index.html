Компиляция программы1

Компиляция программ на C# осуществляется компилятор csc.exe (C Sharp Compiler). Как правило, при использовании Windows XP и .Net Framework v2.0, он находится в папке c:\WINDOWS\Microsoft.NET\Framework\v2.0.50727. Этот путь при установке .Net Framewrok 2 прописывается в переменную среды net. То есть компилятор можно вызвать следующим образом %net%\csc.exe. Компилятор обладает множеством параметров. Укажем только один основной наиболее часто применяемый.

  • /t:exe
  • /t:winexe
  • /t:library
  • /t:module

Параметр /t:exe берется по умолчанию. То есть, чтобы получить из нашего файла исполняемую программу, необходимо ввести следующую команду %net%\csc hello.cs, получим файл hello.exe.
